The following API endpoints are available:

  • /pokemon - Get a list of all Pokemon
  • /pokemon?name=Pikachu - Get the Pokemon with the name "pikachu"
  • /pokemon?id=25 - Get the Pokemon with the ID of 25
  • /pokemon?limit=5 - Get a list of the first 5 Pokemon
  • /pokemon?type=Electric - Get a list of Pokemon with the type "Electric"
