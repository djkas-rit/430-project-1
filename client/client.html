Pokédex API

The following API endpoints are available:

  • /pokemon - Get a list of all available Pokémon
  • /pokemon?name=Pikachu - Get the Pokémon with the name "pikachu"
  • /pokemon?id=25 - Get the Pokémon with the ID of 25
  • /pokemon?limit=5 - Get a list of the first 5 Pokémon
  • /pokemon?limit=5&offset=5 - Get a list of 5 Pokémon starting from the 6th Pokémon
  • /pokemon?type=Electric - Get a list of Pokémon with the type "Electric"
  • /pokemon?sort=desc - Get a list of Pokémon sorted in descending order
